## What shipped

### Hydration mismatch fixes
- `components/parallax-backdrops.tsx`: added `mounted` + `vh` state
  gates. Renders `translate3d(0, 0px, 0)` on server + first client
  render, only diverges after `useEffect` fires.
- `components/video-parallax-hero.tsx`: same gate pattern around the
  `scrollY * parallaxFactor` computation.

### Logo + favicon
- Generated 16/32/180/192/512 PNGs from `/public/brand/logo-mark.png`
  ‚Üí `/public/icons/` + `app/favicon.ico` (replaces the placeholder
  `app/icon.svg`, which was the stylized antlered skull rather than
  the real Buck Mtn brand mark).
- `app/layout.tsx` icons metadata + manifest pointer.
- `public/manifest.webmanifest` so Add-to-home-screen installs.

### Nav fit at mobile width
- `components/site-nav.tsx`: logo `<Link>` got `shrink-0` (Next/Image
  was collapsing to `0px` inside the flex nav). Logo shrunk to
  `h-14 w-14` on mobile, keeping `h-28 w-28` on md+.
- Tightened mobile gap + text size (`gap-0.5 text-[13px]`).
- IG icon hidden below md (still in footer + on every page).

### Hero CTAs
- `app/page.tsx`: hero buttons swap to `.cta-pill` (Sierra-amber
  gradient pill) + `.cta-pill-ghost` (glass). Inner `<span data-arrow>`
  nudges +3px on hover.
- Improves on BMH's flat green pill (no transform on hover) and
  Jackie's pink magnetic pill (too pop for cannabis).

### 3D tile-fall + card tilt on bento grid
- `app/globals.css`: new `.tile-fall` keyframe ‚Äî `perspective(900px)`
  + `rotateX(-28deg)` + `translate3d(0, -60px, -160px)` + blur, all
  scroll-driven via `animation-timeline: view()`. Stagger via
  `--tile-i` (65ms step).
- New `.card-tilt` for hover (CSS-only ¬±2.5¬∞ rotation + lift) and
  `.reveal-blur-up` utility (not wired yet ‚Äî Phase 2).
- `components/bento-strain-grid.tsx`: wires `.tile-fall` +
  `.card-tilt` + `--tile-i` index to each tile. Also added explicit
  `auto-rows-[180px]` for mobile (grid rows were collapsing because
  every tile child is absolute-positioned).
