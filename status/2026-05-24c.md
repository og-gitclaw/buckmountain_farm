## Code shipped

- `lib/session.ts` — single source of truth for `bm_session` cookie parsing
- `components/age-gate.tsx` — CA 21+ overlay, 30-day localStorage TTL,
  reduced-motion safe, never blocks SSR
- `components/push-subscribe-button.tsx` — now wired on `/loyalty`
- Layout: mounts `<AgeGate />` after children
- OAuth state hardening: HMAC-signed nonce (`<nonce>.<hmac>`) using
  `SESSION_SECRET`, verified on the callback (`/api/auth/google` +
  `/api/auth/google/callback`). Falls back to unsigned nonce if
  SESSION_SECRET isn't set (still better than nothing for dev/preview).
- `next.config.ts` `redirects()`: 14 legacy Squarespace + BigCommerce
  paths now 301 to the new IA
- `next.config.ts` `headers()`: `X-Robots-Tag: noindex,nofollow,noimageindex`
  on every response (defense-in-depth on top of the meta robots +
  robots.txt)
- Loyalty claim page now actually fetches `/api/loyalty/claim/[token]`
  via fetch with cookie forwarding, displays credited points
- Homepage footer expanded with About/Contact/Wholesale/COA/Privacy/Terms
- `app/sitemap.ts` extended w/ the new public pages
